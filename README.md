{
	1. 자신의 작업은 자신의 Branch를 만들어서 작업합니다.
	2. 매일 작업 시작 전, Pull 작업을 반드시 실행합니다.
	3. Commit 작업 시에는 반드시 내용을 적습니다.
	4. Merge 작업은 정말 중요합니다.
		★ Pull 작업을 반드시 선행한 후, 오류가 없음을 확신한 했을때만 실행합니다.
	5. 백업버전은 절대! 손대지 않습니다.
